Simple text formatting is supported in text input fields.

|Syntax                   |Result       |
|-------------------------|-------------|
|`**text**`               | bold        |
|`*text*`                 | italic      |
|`***text***`             | bold+italic |
|`H~2~0`                  | subscript   |
|`mc^2^`                  | superscript |
|`code`               | inline code |
|`<https://bla.com>`      | link        |
|`[bla](https://bla.com)` | named link  |
|`\`                      | soft break  |
